# Modern Website

A beautiful, responsive website built with HTML, CSS, and JavaScript featuring modern design principles and smooth animations.

## 🚀 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Contact Form**: Functional contact form with validation
- **Smooth Animations**: CSS animations and JavaScript-powered interactions
- **Performance Optimized**: Fast loading and smooth performance

## 📁 File Structure

```
Website/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## 🛠️ Setup Instructions

1. **Download/Clone** the project files to your local machine
2. **Open** `index.html` in your web browser
3. **That's it!** The website should load immediately

No build process or dependencies required - it's ready to use!

## 🎨 Customization Guide

### Changing Colors
The website uses a modern blue color scheme. To change colors, update these CSS variables in `styles.css`:

```css
/* Primary colors */
--primary-color: #2563eb;
--secondary-color: #667eea;
--accent-color: #764ba2;
```

### Updating Content
- **Hero Section**: Edit the title and subtitle in the hero section of `index.html`
- **About Section**: Update the mission text and statistics
- **Services**: Modify the service cards with your own services
- **Contact Information**: Update email, phone, and location details
- **Footer**: Change the brand name and social media links

### Adding New Sections
1. Add your HTML structure in `index.html`
2. Style it in `styles.css`
3. Add any JavaScript functionality in `script.js`

### Changing Fonts
The website uses Inter font from Google Fonts. To change fonts:

1. Update the Google Fonts link in `index.html`
2. Change the font-family in `styles.css`:

```css
body {
    font-family: 'Your-Font-Name', sans-serif;
}
```
